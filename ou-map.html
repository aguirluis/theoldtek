OU Map Blueprint

Visualizing Active Directory structure for scalable device and user management

Overview

This OU map reflects the current structure of my home lab ADUC, segmented by device type, user role, and service accounts. It supports RBAC enforcement, GPO targeting, and provisioning automation.

Visual Map

Key Highlights

  • Lab Devices: Nested by OS and function (e.g., Linux servers, Windows endpoints)
  • Family Devices: Segmented by VLAN and age group (e.g., Kids VLAN)
  • Service Accounts: Isolated OU with strict GPO lockdown
