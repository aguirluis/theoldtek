AWS Serverless Feedback App

A fully serverless feedback submission workflow built using AWS Lambda, API Gateway, DynamoDB, and CloudWatch.

Architecture Overview

This project demonstrates a clean, event‑driven serverless pattern using AWS managed services:

  • AWS Lambda
  • API Gateway
  • DynamoDB
  • CloudWatch Monitoring

When a user submits the form below, the data is sent to an API Gateway endpoint, processed by a Lambda function, and stored in DynamoDB. CloudWatch provides logging and observability.
