Building an Enterprise-Style ADDS, DHCP, and RAS/NAT Lab with Windows Server 2019

Published: 2025

This project recreates a real enterprise identity and network environment using Windows Server 2019. It includes Active Directory Domain Services, DHCP, and Routing and Remote Access (RAS/NAT).

Active Directory Setup

I deployed AD DS, created OUs, and implemented a clean RBAC model for administrators, helpdesk, and standard users.

DHCP Configuration

DHCP handles IP assignment for multiple VLANs. I configured:

  • Scopes for each subnet
  • Reservations for servers
  • DNS and gateway options

RAS/NAT

RAS provides routing between internal networks and the external network. NAT allows internal VMs to reach the internet without exposing them directly.

Why This Lab Is Useful

  • Perfect for identity management practice
  • Great for cybersecurity testing
  • Replicates real enterprise infrastructure
