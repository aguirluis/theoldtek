Designing a Production-Style Home Lab with Proxmox, pfSense, and TrueNAS

Published: 2025

My Proxmox home lab is the backbone of all my testing, automation, and cybersecurity experimentation. The goal was to build an environment that feels like a real enterprise network — but running entirely at home.

Core Components

  • Proxmox VE for virtualization
  • pfSense for firewalling and routing
  • TrueNAS for storage and NFS/SMB shares
  • Automated VM provisioning using shell scripts

Network Layout

I created segmented VLANs for management, servers, clients, and lab networks. pfSense handles DHCP, DNS forwarding, and firewall rules between segments.

Automation

One of the biggest wins was automating VM creation. Using Proxmox’s CLI tools, I wrote scripts that:

  • Create VMs from cloud-init templates
  • Apply CPU/RAM defaults
  • Attach storage and networks
  • Start and configure the VM automatically

Why This Lab Matters

  • It mirrors real enterprise infrastructure.
  • It’s perfect for cybersecurity and identity management testing.
  • It saves hours of manual VM setup.
