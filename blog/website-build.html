Building TheOldTek: My Custom Portfolio Website on GitHub Pages

Published: 2025

This website is the foundation of my personal brand. I built it from scratch using HTML, CSS, and GitHub Pages — no frameworks, no templates, just clean code and a design inspired by blueprints and pixel art.

Design Goals

  • Fast and lightweight
  • Easy to maintain
  • Professional but unique
  • Dark/light theme toggle

Sidebar Layout

The site uses a three-section sidebar: profile, navigation, and social links. This keeps navigation consistent across all pages.

Project Cards

Each project is displayed as a responsive card with hover effects and clean typography. The cards link to project pages, blog posts, and GitHub repos.

Why GitHub Pages?

  • Free hosting
  • Automatic HTTPS
  • Perfect for static sites
  • Easy version control

What I Learned

  • How to design a cohesive personal brand
  • How to build a responsive layout from scratch
  • How to structure a multi-page static site cleanly
